프로그래머스 모의 테스트

sql및 마지막 문제들은 다 잘 모르는 부분이였다.
알고리즘은 첫 번째 문제 말고는 다 쉬운 편… 첫번째 문제도 코딜리티에서 나왔던 문제여서 금방 풀 수 있었다.
길이가 n인 배열에 1부터 n까지 숫자가 중복 없이 한 번씩 들어 있는지를 확인하려고 합니다.
1부터 n까지 숫자가 중복 없이 한 번씩 들어 있는 경우 true를, 아닌 경우 false를 반환하도록 함수 solution을 완성해주세요.

제한사항
배열의 길이는 10만 이하입니다.
배열의 원소는 10만 이하의 자연수입니다.

이거 다 뭔 말인지 공부해볼 것 !
①
Shared memory 는 shared memory 를 사용하는 모든 프로세스가 종료되면 커널이 자동으로 공간을 삭제한다.

②
세마포어는 프로세스간 동기화를 위한 IPC 이다.

③
FIFE와 pipe의 차이점은 이름이 있는 파일을 통해서 통신을 할 수 있다는 점이다.

④
FIFO는 queue 방식의 입/출력을 지원한다.

⑤
pipe를 이용하면 2개의 file descriptor를 생성할 수 있다. 하나는 읽기 전용, 다른 하나는 쓰기전용의 파이프로 사용된다.
